# Naruto(Vue 3 + TypeScript + Vite)

#### 🌈 介绍

基于 vue3.x + typescript + vite + element plus + pinia

#### ⚡ 使用说明

建议使用 cnpm。<a href="http://nodejs.cn/" target="_blank">node 版本 > 12xx.xx.x</a>

```bash
# 克隆项目
git clone https://github.com/crazyone2one/naruto.git

# 进入项目
cd naruto

# 安装依赖
cnpm install

# 运行项目
cnpm run dev

# 打包发布
cnpm run build
```
